Triển khai metric server:
- Metric server trong kubernets giám sát tài nguyên sử dụng trên cluster, cung cấp các API để các thành phần khác truy vấn đến biết được và mức độ sử dụng tài nguyên (CPU, Memory) của pod, node,...
- Cần có metric server để HPA hoạt động chính xác

Để triển khai metric server, lấy mã nguồn từ github:
  `` git clone git@github.com:kubernetes-sigs/metrics-server.git``
